# 2阶魔方游戏

一个基于Pygame的3D 2阶魔方游戏。

## 环境要求

- Python 3.9 或更高版本
- Pygame 2.1.0 或更高版本

## 安装步骤

1. 进入项目目录：
```bash
cd main
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 运行游戏

```bash
python game.py
```

## 游戏说明

### 基本操作

1. **视角旋转**：在空白区域按住鼠标左键并拖动，可以360度旋转查看魔方

2. **选择层**：
   - 鼠标左键点击魔方上的任意方块，激活该方块所在的行
   - 双击同一位置（300ms内），切换为列选择模式
   - 再次双击可切换回行选择模式
   - 被选中的层会高亮显示

3. **旋转层**：选中层后，点击鼠标右键即可旋转该层

4. **打乱魔方**：点击右侧面板的"打乱魔方"按钮，系统会随机打乱魔方

5. **重置魔方**：点击右侧面板的"重置魔方"按钮，将魔方恢复到初始已还原状态

### 颜色说明

- 白色 (W) - 顶面 (U)
- 黄色 (Y) - 底面 (D)
- 绿色 (G) - 前面 (F)
- 蓝色 (B) - 后面 (B)
- 橙色 (O) - 左面 (L)
- 红色 (R) - 右面 (R)

### 游戏目标

将打乱后的魔方还原，使每个面的所有方块颜色一致。

## 项目结构

```
main/
├── game.py           # 游戏入口和主循环
├── requirements.txt  # 依赖文件
├── README.md        # 说明文档
└── kernel/
    ├── check.py     # 魔方状态管理、打乱逻辑、可解性检查
    ├── ui.py        # UI界面组件、按钮、控制面板
    └── control.py   # 3D渲染、鼠标交互、动画效果
```

## 技术特性

1. **3D渲染**：使用简单的透视投影实现3D效果
2. **流畅动画**：层旋转时有平滑的动画效果
3. **多线程**：打乱操作在后台线程执行，不影响UI响应
4. **中文支持**：自动检测系统中文字体，支持中文显示
5. **可解保证**：打乱算法确保生成的魔方一定可以还原

## 操作提示

- 点击空白区域拖动来改变视角
- 点击魔方面块来选择行/列
- 双击切换行/列选择模式
- 右键旋转选中的层
- 成功还原后会显示恭喜提示
